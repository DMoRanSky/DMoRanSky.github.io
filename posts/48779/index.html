反蛇形矩阵

1  2  6  7  15  16
3  5  8  14 17  26
4  9  13 18 25  27
10 12 19 24 28  33
11 20 23 29 32  34
21 22 30 31 35  36

已知行数i，列数j，需推出values[i][j]。
这个矩阵是由可以看做从左下~右上为一组(如4~6,10~7)，都为连续自然数，只不过遇到边界前往下一列。
首先我们发现values[i][j]由两部分组成：
前面斜列的总和sum与在此斜列的项数q
首先我们推一下前面斜列的总和，我们首先要知道我们在第几个斜列p。
很明显，p可以是此斜列在第一行数的列数-1,
例如：13，位于（11~15斜列），15的列数为5，故由在这之前4个斜列。
推到第一行的数很简单，原本的i值每上升一格，j值便向右一格，到i=1停止，故：

p=i+j-1

sum因为是1+2+…+p-1，我们可以用等差数列的公式来做

sum=(p-1)*p/2

但是，sum并不一定是完全的！比如在(22~26)这一列，p>n，所以根本不存在，也就不存在计数了：

if(p>n)sum-=(p-n+1)*(p-n);

下面，q如何求？
观察一下，如果p是偶数，不就是i吗？如果p是奇数，不就是反向求法，p-i+1吗？
不！还有一个特殊判断，因为这斜列可能不是在i=1时就开始计数的，如果p>n，要忽略掉p-n个数字：

if(p%2==1)q=p-i+1+(p>n?p-n:0);
else q=i+(p>n?p-n:0);

接下来把sum+q输出就行啦，整个公式就推好了。

#include<cstdio>
using namespace std;
int main(){
    int n;
    scanf("%d",&n);
    for(int i=1;i<=n;i++){
        for(int j=1;j<=n;j++){
            int p=j+i-1;
            int sum=(p-1)*p/2;
            if(p>n)sum-=(p-n+1)*(p-n);
            printf("%d ",(p%2==1)?sum+p-i+1+(p>n?p-n:0):sum+i+(p>n?p-n:0));
        }
        printf("\n");
    }
    return 0;
}

        }

    }

}
