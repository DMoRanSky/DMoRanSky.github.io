代码高亮

不喜欢hexo原版的highlight。
使用了hexo-prism-plugin 的 Hexo 插件来做代码高亮，安装命令如下：

npm i -S hexo-prism-plugin

之后，修改 Hexo 根目录下 _config.yml 文件中 highlight.enable 的值为 false，并新增 prism 插件相关的配置，主要配置如下：

highlight:
  enable: false

prism_plugin:
  mode: 'preprocess'    
  theme: 'tomorrow'
  line_number: false    
  custom_css:

博客部署

用作将本地博客部署git部署到库，hexo-deployer-git，安装命令如下：

npm install hexo-deployer-git --save

集成后台

安装后，可在localhost:4000/admin访问集成后台，进行撰写文章。
npm install –save hexo-admin
安装命令如下：

npm install --save hexo-admin

内容搜索

使用到了 hexo-generator-search 的 Hexo 插件来做内容搜索，安装命令如下：

npm install hexo-generator-search --save

具体应用请见链接。

博客结构

使用了hexo-abbrlink，以达到将post转为唯一id的目的，方便进行SEO、而且中文链接过于难看，安装命令如下：

npm install hexo-abbrlink --save

在根目录下的_config.yml换博客结构：

permalink: posts/:abbrlink/

统计文章字数、阅读时长

hexo-wordcount可达到此目的，安装方式：

npm i --save hexo-wordcount

具体应用请见链接。

RSS 订阅

hexo-generator-feed 可做RSS，安装命令如下：

npm install hexo-generator-feed --save

在 Hexo 根目录下的 _config.yml 文件中，新增以下的配置项：

feed:
  type: atom
  path: atom.xml
  limit: 20
  hub:
  content:
  content_limit: 140
  content_limit_delim: ' '
  order_by: -date
